# Инструкция по установке Votobu

## Быстрая установка (Windows)

### Вариант 1: Автоматическая установка

1. Убедитесь, что у вас установлен Python 3.8 или выше
2. Дважды кликните на файл `install.bat`
3. Дождитесь завершения установки
4. Запустите приложение через `run.bat`

### Вариант 2: Ручная установка

1. Откройте командную строку в папке проекта

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Создайте иконки:
```bash
python create_icons.py
```

4. Запустите приложение:
```bash
python src/main.py
```

## Системные требования

- **Операционная система:** Windows 10/11 (Linux/Mac поддерживаются частично)
- **Python:** 3.8 или выше
- **RAM:** минимум 2 GB (рекомендуется 4 GB для моделей medium/large)
- **Микрофон:** любой подключенный микрофон
- **Интернет:** НЕ требуется (Whisper работает локально)

## Первый запуск

1. После запуска приложения появится иконка в системном трее (область уведомлений)
2. При первом запуске произойдет загрузка модели Whisper (может занять несколько минут)
3. После загрузки модели появится уведомление "Votobu запущен"

## Базовое использование

1. **Начать запись:** Зажмите горячую клавишу (по умолчанию F9)
2. **Говорите:** В микрофон четко произносите текст
3. **Остановить запись:** Отпустите горячую клавишу
4. **Ожидание:** Подождите несколько секунд (распознавание)
5. **Результат:** Текст автоматически скопируется в буфер обмена
6. **Вставка:** Нажмите Ctrl+V в любом приложении

## Настройка

Кликните правой кнопкой мыши на иконке в трее → **"Настройки"**

### Горячая клавиша
- Нажмите кнопку "Изменить клавишу"
- Нажмите желаемую клавишу (F1-F12, буквы, цифры)
- Сохраните настройки

### Язык распознавания
- **Русский** - для русской речи (быстрее)
- **Английский** - для английской речи (быстрее)
- **Автоопределение** - определяет язык автоматически (медленнее)

### Модель Whisper
- **tiny** - самая быстрая, низкое качество
- **base** - хороший баланс (по умолчанию, рекомендуется)
- **small** - лучше качество, медленнее
- **medium** - высокое качество, требует больше RAM
- **large** - максимальное качество, очень медленная

## Решение проблем

### Микрофон не работает
- Убедитесь, что микрофон подключен и выбран по умолчанию в системе
- Проверьте настройки конфиденциальности Windows (Настройки → Конфиденциальность → Микрофон)

### Модель не загружается
- Проверьте подключение к интернету при первой загрузке
- Убедитесь, что достаточно свободного места на диске (минимум 1 GB)
- Попробуйте модель меньшего размера (tiny или base)

### Плохое качество распознавания
- Говорите четче и ближе к микрофону
- Уменьшите фоновый шум
- Попробуйте модель большего размера (small или medium)
- Укажите конкретный язык вместо автоопределения

### Приложение не запускается
- Убедитесь, что все зависимости установлены: `pip install -r requirements.txt`
- Проверьте версию Python: `python --version` (должна быть 3.8+)
- Проверьте логи в консоли при запуске через `python src/main.py`

## Деинсталляция

1. Закройте приложение (ПКМ на иконке → Выход)
2. Удалите папку проекта
3. Удалите конфигурацию: `%APPDATA%\Votobu`
4. (Опционально) Удалите зависимости: `pip uninstall -r requirements.txt -y`

## Дополнительная информация

### Где хранятся настройки?
`%APPDATA%\Votobu\config.json` (Windows)
`~/.config/votobu/config.json` (Linux/Mac)

### Где хранятся временные аудиофайлы?
Во временной папке системы, автоматически удаляются после распознавания

### Как обновить приложение?
1. Загрузите новую версию
2. Скопируйте файлы поверх старых
3. Настройки сохранятся автоматически

## Поддержка

Если у вас возникли проблемы:
1. Проверьте раздел "Решение проблем" выше
2. Проверьте консольные логи при запуске через `python src/main.py`
3. Создайте Issue на GitHub с описанием проблемы и логами
